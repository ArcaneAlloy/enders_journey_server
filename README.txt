Below is a list of commonly asked questions. If you have a question not answered in this document, please ask the question in our Discord: https://discord.gg/9S5tfrY4Fr

-- How do I start the server? --
First YOU NEED JAVA 8 TO PLAY Ender's Journey Modpack, THIS CAN BE DOWNLOADED HERE: https://www.java.com/en/download/ie_manual.jsp (install and run the .exe file)
Once installed just open the start.bat file

-- How do I join my server? --
1- Install the modpack using a modded launcher for example Curseforge, GDLauncher, ATLauncher, Multicraft, etc
2- Go to the Multiplayer tab in the main menu
3- Click Direct Connect and type localhost

-- How do others join my server? --
They will connect using your Public IPv4 Address, which can be found here: https://whatismyipaddress.com/
If people are unable to join with this IP, you will have to follow a Portforwarding guide, I attach you an example here: https://www.noip.com/support/knowledgebase/general-port-forwarding-guide/
Be ensure that your firewall or Antivirus are not blocking the connections to your server

-- How do I allocate more RAM? --
1- Double click the start.bat and open it with a Text Editor, for example, Notepad
2- Modify the value of MAX_RAM in the line 4 with the number or GB of your RAM that you want to assign

-- How improve the perfomance of the server? --
You can preload the chunks to reduce the CPU load, here are a few mods that you can add to your server to do this: 
- https://www.curseforge.com/minecraft/mc-mods/chunkpregenerator
- https://www.curseforge.com/minecraft/mc-mods/chunky-pregenerator-forge/files 

Other option it is reduce the "View Distance" value in the server.properties file: https://shockbyte.com/billing/knowledgebase/134/Adjusting-View-Distance-and-Simulation-Distance-on-Your-Minecraft-Server.html
